# KAZAN-GAICA

Проект для разработки бота под соревнование GAICA 

Сейчас репозиторий содержит базовую структуру проекта, которую команда может постепенно заполнять своей логикой: сетевым клиентом, парсером состояния игры, стратегией и тактикой бота.

## Структура проекта

- `main.py` - точка входа для запуска бота.
- `gaica_bot/` - пакет для основной логики бота.
- `gaica_bot/client.py` - заготовка под TCP-клиент и обмен JSON-сообщениями.
- `gaica_bot/controller.py` - заготовка под главный игровой цикл и принятие решений.
- `gaica_bot/game_state.py` - заготовка под разбор состояния матча и снапшотов.
- `gaica_bot/geometry.py` - заготовка под геометрию, расстояния и проверки столкновений.
- `gaica_bot/models.py` - заготовка под модели данных и структуры команд.
- `gaica_bot/strategy.py` - заготовка под стратегию и эвристики.
